* `ipconfig getifaddr en0` returns the ip address for my wifi connection. en1 would return a wireless connection.
* `nslookup [domain name]` can be used to find a domain's IP address
* Similarly, `nslookup [ip address]` can be used to find the corresponding domain name
* scp [file name] [username]@[ip address]:[directory path] will copy a file to another computer over the network.
* using `lsof -i :[port number]` allows you to get processes listening to that port
* `lsof -i -P -n | grep LISTEN` gives you all of the ports that are open on the device
* using `curl -o [location on local disk] [url]` you can save a webpage
* using `curl -v [url]` will show you the entire request, including the response headers beginning with `<`
* using `curl -v [url]` will show you the entire request, including the request headers beginning with `>`
